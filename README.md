# 通讯录管理系统 - 极限编程课程作业


## 项目概述
本项目是极限编程课程的团队作业，旨在开发一个具备**联系人收藏、多联系方式管理、Excel导入导出**等核心功能的通讯录管理系统，同时完成云端部署（可选）、团队协作开发与技术博客撰写等任务。

## 功能列表
### 核心基础功能（必选）
1. **联系人收藏/书签功能（25分）**
   - 用户可标记联系人为“重要/常用”
   - 提供收藏联系人的快速访问入口
2. **多联系方式添加功能（10分）**
   - 支持为单个联系人添加手机号、邮箱、社交账号、物理地址等多种联系方式
   - 支持联系方式的编辑、删除与展示
3. **Excel导入导出功能（25分）**
   - **导出**：将所有联系人信息按行导出至Excel，每类信息对应单独列（姓名、电话、邮箱等）
   - **导入**：读取符合格式的Excel文件，将联系人信息批量导入系统
4. **页面样式（10分）**
   - 简洁美观的用户界面，支持功能的直观操作

## 开发与协作流程
### 1. 仓库协作
- 仓库地址：(https://github.com/dhufisuifiuaf/832302114_832302104.git)
- 协作规范：
  1. Fork主仓库后，基于`dev`分支创建功能分支（如`feature/bookmark`）
  2. 代码提交遵循规范（如`feat: 实现联系人收藏功能`）
  3. 功能完成后发起Pull Request，团队评审后合并至`dev`分支
  4. 最终发布版本时，从`dev`分支合并至`main`分支

### 2. 代码规范
- 命名规范：变量/函数使用小驼峰，类使用大驼峰
- 注释规范：核心函数、复杂逻辑添加中文注释
- 提交信息规范：`类型: 描述`（类型包括feat/fix/docs/style等）
